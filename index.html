Application 6: Real-Time ESP32 Demo

This page shows a short demo of my Application 6 project for EEL 4742C Real-Time Systems at UCF. The goal of the project is to build a small real-time system on the ESP32 that reacts to inputs and updates outputs on a strict schedule.

Video Demo

Project Overview

In this project the ESP32 runs multiple tasks that check inputs, run control logic, and update outputs. The main idea is to keep the system responsive while still meeting timing deadlines. From a user’s point of view, you can see [briefly say what changes: LEDs, serial output, etc.] as the system reacts in real time.

Tech Summary

  • Platform: ESP32
  • OS: FreeRTOS
  • Language: C / C++
  • Key features: periodic tasks, interrupts, queues for communication
